# Simple Temperature Converter
## How it works?
It uses simple mathematics formulas for converting temperature values from one measurement system to another measurement
system.

You can convert from Fahrenheit to Celsius, from Kelvin to Fahrenheit, from Celsius to Kelvin and so on.

## How to run the program?
1. You have to install Java 21 (or newer).
2. You need to download Main.java. Open Terminal or any other Command-Line app and open the directory in the command-line
to the place you have the Main.java saved.
3. In the command-line type "javac Main.java" to compile the program.
4. Now you type "java Main" to start the program.
